## [673. 最长递增子序列的个数](https://leetcode-cn.com/problems/number-of-longest-increasing-subsequence/)

#### 关键字：【动态规划】

#### 难度：中等

> **题目描述**
>
> 给定一个未排序的整数数组，找到最长递增子序列的个数。

> 
>
> **Example:**
>
> ```
> 输入: [1,3,5,4,7]
> 输出: 2
> 解释: 有两个最长递增子序列，分别是 [1, 3, 4, 7] 和[1, 3, 5, 7]。
> ```

> ```
> 输入: [2,2,2,2,2]
> 输出: 5
> 解释: 最长递增子序列的长度是1，并且存在5个子序列的长度为1，因此输出5。
> ```


### 解法

- [x] 动态规划
- [ ] 线段树

### 代码

```java
// 解法一
class Solution {
    public int findNumberOfLIS(int[] nums) {
        int len = nums.length;
        if(len == 0) return 0;
        int[][] dp = new int[len][2]; // {第i个元素的最大长度， 组成这个长度的个数}
        dp[0][0] = 1;
        dp[0][1] = 1; 

        for(int i = 1; i < len; i++){
            int count = 1;
            int max_length = 1;

            for(int j = 0; j < i; j++){
                if(nums[i]>nums[j]){
                    int max_tmp = dp[j][0] + 1;
                    if(max_tmp>max_length){
                        max_length = max_tmp;
                        count = dp[j][1];
                    }else if(max_tmp == max_length){
                        count += dp[j][1];
                    }
                }else if(nums[i] == nums[j]){
                    max_length = dp[j][0];
                    count = dp[j][1];
                }
            }

            dp[i][0] = max_length;
            dp[i][1] = count;
        }

        int count = 0;
        int max = 1;
        for(int i = 0; i < len; i++){
            if(dp[i][0] > max){
                count = dp[i][1];
                max = dp[i][0];
            }else if(dp[i][0] == max){
                count += dp[i][1];
            }
        }

        return count;


    }
}
```
